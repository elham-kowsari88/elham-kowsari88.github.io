Editing Guide

Your website content is stored in simple Markdown files inside the content/ folder. Here's how to edit them:

How It Works

  1. Select a file from the sidebar on the left
  2. Edit the text in the editor — you'll see a live preview on the right
  3. Download the file and replace it in your GitHub repository
  4. Your website will automatically update!

File Structure

Each file has two parts:

---
title: Page Title
subtitle: A short description
---

Your content goes here in **Markdown** format.

Markdown Cheat Sheet

What you typeWhat you get
# Heading 1Large heading
## Heading 2Medium heading
### Heading 3Small heading
**bold text**bold text
*italic text*italic text
[Link text](https://url.com)A clickable link
![Alt text](image.jpg)An image
- List itemBullet point
1. Numbered itemNumbered list
---Horizontal line (separator)
`code`Inline code

Adding a New Publication

Open publications.md and add under the right year:

- **Your Paper Title Here**
  - *E. Kowsari*, Co-Author Name
  - *Journal or Conference Name*, Year
  - DOI: [10.xxx/xxx](https://doi.org/10.xxx/xxx)
  - `journal` `2026`

Adding a New Project

Open projects.md and add at the end (before the last ---):

---

## Your Project Title (Year)

- **Status:** Published / Under Review
- **Venue:** Conference or Journal Name
- **Authors:** Author 1, Author 2

Description of your project...

### Figures

![Description](your-image.jpg)

### Links

- [Paper](https://link-to-paper.com)

Updating Your Photo

  1. Upload your photo to the assets/img/ folder
  2. In hero.md, change the photo: line to your filename
